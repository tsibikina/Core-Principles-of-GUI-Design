# SQL Database Browser - это полнофункциональное desktop-приложение на PyQt5 для просмотра и анализа SQLite баз данных.
Приложение предоставляет интуитивно понятный графический интерфейс для выполнения SQL-запросов и визуализации структуры базы данных.

## Функциональность
Управление соединением
Set connection - установка соединения с SQLite базой данных

Close connection - закрытие соединения и очистка интерфейса

### Элементы управления

**QComboBox:** Выбор колонки для просмотра в формате table.column

**Кнопки запросов:** Специализированные запросы к разным аспектам БД

**Информационная панель:** Отображение текущего файла БД

**Адаптивный интерфейс:** Элементы активируются только при активном соединении

#### Архитектура приложения
**1. Класс DatabaseManager**

Назначение: Централизованное управление операциями с БД

class DatabaseManager:

    def connect(db_path)          # Установка соединения
    
    def close()                   # Закрытие соединения  
    
    def execute_query(query)      # Выполнение SQL-запросов
    
    def get_table_names()         # Получение списка таблиц
    
    def get_table_columns()       # Получение колонок таблицы
    
    def get_all_columns_from_all_tables()  # Все колонки всех таблиц
    
    
**2. Класс MainWindow**

Назначение: Управление графическим интерфейсом


class MainWindow(QMainWindow):

    def init_ui()                 # Инициализация интерфейса
    
    def set_connection()          # Обработчик подключения к БД
    
    def close_connection()        # Обработчик отключения от БД
    
    def on_column_selected()      # Обработчик выбора колонки
    
    def execute_*_query()         # Обработчики кнопок запросов
    
    def display_data_in_table()   # Отображение данных в таблицах
    
    
**3. Компоненты интерфейса**

QTabWidget: 5 вкладок для различной информации

QTableWidget: Отображение результатов запросов в табличном виде

QComboBox: Интеллектуальный выбор колонок из всех таблиц

QPushButton: Управляющие кнопки с логикой активации/деактивации

##### Установка и запуск
**Требования:**

pip install PyQt5

**Запуск приложения:**
Создание тестовой БД (опционально):

python create_test_db.py

**Запуск основного приложения:**

python main_app.py

### Использование
**Пошаговая инструкция:**
- Запустите приложение - откроется главное окно с неактивными элементами

- Нажмите "Set connection" - выберите файл SQLite базы данных

- Автоматическая загрузка - Tab1 заполнится схемой базы данных

**Исследуйте данные:**

- Нажмите "Show Table Names" для списка таблиц (Tab2)

- Выберите колонку из выпадающего списка для просмотра данных (Tab3)

- Нажмите "Show Tables Data" для образцов данных всех таблиц (Tab4)

- Нажмите "Show Table Info" для структурной информации (Tab5)

- Закройте соединение - кнопка "Close connection" для очистки
